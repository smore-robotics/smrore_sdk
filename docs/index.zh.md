## 仓库内容

- [x] 带版本号的 C++ SDK Release 压缩包与 Python wheel
- [x] 可运行的 C++ 示例（`examples/`）与 Python 示例（`examples_py/`）
- [x] 下载最新 C++ SDK 和构建示例的脚本

该仓库不包含 SDK 源码。用户从 GitHub Releases 下载带版本号的 SDK 制品，并基于
这些制品构建自己的应用。
