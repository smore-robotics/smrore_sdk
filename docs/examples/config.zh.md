# 配置示例

## 运行前

配置示例会**读取或临时修改**机器人配置，并在结束时尽量恢复原值。请先看终端打印
出的原始值，确认无误后再继续。`config_limits` 与 `payload` 在恢复失败时会以非零码
退出，提醒你手动检查。

## config_limits

### 作用

读取关节限位等配置，临时收紧后再恢复为原值。

### 适用场景

- 查看当前限位/速度等配置。
- 了解如何安全地临时修改并恢复配置。

### 完整源码

```cpp
--8<-- "examples/config/config_limits.cpp"
```

## waypoints

### 作用

新增一个带专用名（`smrcore_example_demo_home`）的关节路点、列出、再删除该路点。
若同名路点已存在则拒绝运行，避免覆盖你的数据。

### 适用场景

- 管理命名路点（增/查/删）。

### 完整源码

```cpp
--8<-- "examples/config/waypoints.cpp"
```

## payload

### 作用

读取并打印当前末端负载，设置一个示例负载，结束时恢复为原始负载（原本无负载则清除）。

### 适用场景

- 设置/读取末端工具负载。
- 了解如何在不破坏用户配置的前提下临时改动负载。

### 完整源码

```cpp
--8<-- "examples/config/payload.cpp"
```
